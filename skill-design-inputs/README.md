# Skill Design Inputs

Use this folder to store the source artifacts that will shape the first V2 skills.

## Folders

- `manual-testcase-regulation/` -> regulation, standard, or template for manual testcases
- `manual-testcases-good/` -> good manual testcase examples
- `playwright-good/` -> good Playwright examples
- `playwright-bad/` -> bad or problematic Playwright examples
- `failures/` -> failed test reports and diagnosis examples

## Rule

Prefer one artifact per file.
Keep examples focused.
Redact anything sensitive before committing.
